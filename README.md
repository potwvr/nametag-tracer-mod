# Nametag Tracer

A Fabric mod for Minecraft 1.21.1 that renders tracers and hitboxes to mobs with nametags.

## Features

- **Red Tracers**: Draws lines from your crosshair to the center of mobs with nametags
- **Hitbox Rendering**: Displays a red filled box around the mob's hitbox
- **Wall Penetration**: Both tracers and hitboxes are visible through walls
- **Simple and Lightweight**: Minimal performance impact

## Installation

1. Install [Fabric Loader](https://fabricmc.net/use/)
2. Install [Fabric API](https://modrinth.com/mod/fabric-api)
3. Download the mod JAR from releases
4. Place it in your `.minecraft/mods` folder
5. Launch Minecraft with the Fabric profile

## Building

```bash
./gradlew build
```

The compiled JAR will be in `build/libs/`
